# Spring Boot + Angular2 + Swagger

# Prerequisites

* JDK8
* [node + npm](https://nodejs.org/) (Current Version)
* [Angular CLI](https://cli.angular.io/)

# Run

```
$ cd backend
$ ./gradlew bootRun
```
Open following URL by Web browser.  
http://localhost:8080/

# Run frontend dev server

```
$ cd frontend
$ npm start
```

Open following URL by Web browser.  
http://localhost:4200/

swagger で吐き出したコードが、baseUrl の指定を強制する。
