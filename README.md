## Prerequisites

* JDK8
* [node + npm](https://nodejs.org/) (Current Version)

## Run

```
$ cd backend
$ ./gradlew bootRun
```
Open following URL by Web browser.  
http://localhost:8080/

## Run frontend dev server

```
$ cd frontend
$ npm install
$ npm start
```

Open following URL by Web browser.  
http://localhost:4200/
